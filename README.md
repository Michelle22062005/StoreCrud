# CRUD Activity - StoreCrud

##  Project description
This project consists of an **online clothing store**, where users can:
- View products organized by **categories**
- Explore different **brands**
- Check details such as **color, size, and price**
- Add products to the **shopping cart**
- View the **total price** of the selected products
